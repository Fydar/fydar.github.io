Work Experience

TT Games

Brighton July 2018 - Present

Responcibilities

Server

The majority of my work at TT Games was on the servers. I was responcible for the Clans and Leaderboard database and infrastructure, Receipt Validation, Data Delivery, AB Testing, and many other aspects of the servers.

The server programming introduced me to AspNet Core and MVC, which I utilised on the main game servers and many of the tools that we created to maintain the game.

Client

On the client I worked on extending Unity's UI (including but not limited too support for the IPhone X notch), various tools to increase productivity (such as automated scene cleanup, configuration and profile management) and restful server communication.

Many of the server systems I worked on put me in a good position to also do the client code. For Clans I created the UI and for Data Delivery I created a system to manage the flow of network communications in order to facilitate delivered game data.

Eneme Entertainment

Remote June 2017 - June 2018

I started working with Eneme Entertainment after my first year at Northbrook.

Responcibilities

I worked on many of the core systems that made up the project Eitr. Being an RPG game, these included:

  • Item System
  • Inventories
  • Skill Trees
  • Character Stats
  • Buffs and Debuffs

Modularity and Tooling of the item system was very important. I created multiple ways of representing an item on Eitr, however the most successful was a solution I called "Behaviour Graph", which is heavily featured in my personal project RPGCore.

Behaviour Graph features a fully node based editor that allowed for advanced customization of items.
